Welcome to DevOps InControl

This guide walks you through every screen in the DevOps InControl Dashboard. Pick a topic below or use the navigation on the left.

Getting Started

First-Time Setup

Walk through the setup wizard to connect DevOps InControl to your Azure DevOps organisation.

Settings

Manage your password, Azure DevOps connection, database servers, and email sender.

Daily Monitoring

Dashboard

Your central overview. See which projects have issues and run checks with one click.

Viewing Flagged Issues

Drill into flagged work items, tag them, copy lists, or send reminder emails.

Working with Tags

View, rename, or remove tags attached to work items in your projects.

Pull Requests

PR Monitor

Overview of active pull requests across all your projects with at-a-glance flag badges.

PR Details

Filter, sort, and inspect individual pull requests for a project.

Database Cleanup

DB Monitor

Discover all databases on your configured servers.

DB Project — Cleanup Tracking

Track which databases can be cleaned up based on linked ticket status.

Sprint Planning

Sprint Manager

Create new sprints and populate them with template work items automatically.

Template Manager

Create, delete, copy, and migrate work item templates across projects and teams.

Velocity & Capacity

View and edit team capacity, mark days off, and track velocity.

Portfolio

Roadmap

Portfolio-level timeline of Epics and Features across projects. Drag to reschedule, link dependencies, and push changes to Azure DevOps.

Team Insights

Developer Assessment

Analyse PR activity, code reviews, and work item completion per team member.

Security

Permissions

Permission matrix, person group lookup, and automated security audits.

Build & Release

Pipelines & Releases

View build pipeline runs and release deployment history with status colours.

Configuration

Managing Projects & Checks

Add, edit, or remove monitored projects and choose which checks to enable.

# DevOps InControl
A control and management application for Microsoft Azure Devops

Made with love by our Scrum Master Berend using our AI knowledge.

Feature list:
Sprint populator for stories and all underlying tasks (sprint templates)
Tag overview with unused-tag filtering and instant cleanup in 1 click
Development database detection based on linked ticket activity
PR monitor for ready, stale, no-reviewer, and fast search
Quick team capacity assignment across the sprint
Management-ready development assessment overview

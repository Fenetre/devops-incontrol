# Version 1.0.0 — Initial release

Welcome to DevOps InControl! Here's what you can do:

## Backlog checks
Automatically detect stale work items, missing assignments, orphaned items, and overdue work across your projects.

## Pull request monitor
Keep track of all open pull requests across your projects. See at a glance which PRs need attention, are getting stale, or are waiting for reviews.

## Database project checks
Verify that database projects follow naming conventions and have proper ticket linkage.

## Sprint velocity
Track your team's velocity per sprint, including capacity calculations and projections for future sprints.

## Pipelines & releases
View recent pipeline runs and release deployments for each project at a glance.

## Developer assessment
Get insights into pull request activity, review quality, and cycle times to help your team improve.

## Easy capacity setup
Quickly set up and manage sprint capacities for your team without navigating through Azure DevOps.

## Tag overview
See all tags used across your projects in one place and spot inconsistencies.

## Sprint populator
Automatically populate sprints with work items to save time during sprint planning.

## Permission checks
Automatically check for missing security groups and misconfigured repositories. Browse team and member permissions across all your Azure DevOps projects.

## Settings & configuration
Set up your PAT, configure projects with the setup wizard, and manage email notifications — all from one place.

# What's new in version 1.1.0

## Velocity calculation fix
Fixed an issue where sprint velocity showed incorrect numbers in certain cases.

## Hide sprints in velocity view
You can now hide individual sprints from the velocity chart so they don't skew your averages.

## Assign parents from orphan check
Orphaned work items now have an "Assign Parent" option that shows only the parent types allowed by your DevOps process configuration.

## Release notes
You can now view release notes at any time by clicking the version number in the bottom-left corner of the dashboard.
